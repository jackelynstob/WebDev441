Painting With Emotions

The biggest project I have worked on in Media Arts was in the Fall of 2018. My final project in my Interactive Arts class took a dive into art installations which allow the spectator to interact with it. My group's project Painting With Emotions used software called Affectiva to capture the user's macroexpressions during their viewing of a video. The data we collected was then used to create a linear graph using different colors associated with different emotions.
